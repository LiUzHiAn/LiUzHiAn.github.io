Okay，闲话不多说，今天写的也是一个比较简单的题目，LeetCode-2-两数相加。

题目内容

描述

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

思路

这题还是挺简单的，主要想法就是：

  • 将两个链表左对齐，对应位的数字相加，如果超过相加和超过10，就将超过10的部分向前进位。

  • 这个题目主要就是考虑一下长度的问题。左对齐后可能L1和L2一样长，也可能不一样长，我们就一直加到其中一个链表加到了底（也就是当前结点为None的情况）。再把剩下的相加就OK。

  • 最后就是，不要忘记把最后一步的进位（carry）判断一下，有可能出现（2 -> 4 -> 6) + (5 -> 6 -> 3 -> 9)这种情况。

要注意的地方

  1. 新建一个链表的时候记得要定义两个指针变量，一个用于最后返回整个链表，另一个用来中间移动。
  2. 不断将 sum_/10 的时候要int()包裹一下
  3. 最开始 head=p=ListNode(-1)这句就是链表头结点，为了方便处理而已，这个-1的值是任意给的。
